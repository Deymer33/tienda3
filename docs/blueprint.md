# **App Name**: SYS Govil Information Hub

## Core Features:

- Company Information Display: Show all information for SYS Govil, pulled directly from the text provided.
- Services Overview: Display a comprehensive list of services offered by SYS Govil including abastecimiento y suministros especializados, servicios de mantenimiento industrial y operativo, gestión logística y distribución, asesoría técnica y soporte continuo, proyectos a medida para sector privado y público
- Contact Information: Provide easy access to contact details, including address (Plan de Ayala #2173, Privada Ntra. Sra. de la Soledad, Soledad de Graciano Sánchez, San Luis Potosí 78430), phone number, WhatsApp link, email, and social media links (Facebook and Instagram).
- Inquiry Form: Allow users to submit inquiries about services directly through the app.
- Generative FAQ: Employ a tool using generative AI to analyze the provided company information and address user questions.

## Style Guidelines:

- Primary color: Deep Blue (#0404E2), reflecting the brand's identity, reliability, and professionalism.
- Secondary color: Very light blue (#E8E8FF), providing a clean and professional backdrop to ensure comfortable readability and a modern feel.
- Accent color: Soft Purple (#4747FF), used for interactive elements and highlights to guide user attention and maintain visual interest without overwhelming the deep blue.
- Font pairing: 'Space Grotesk' (sans-serif) for headlines and 'Inter' (sans-serif) for body text, creating a balance between modern aesthetics and readability.
- Use professional and clear icons that align with SYS Govil's services. Icons should be simple and easy to understand.
- Maintain a clean and organized layout to ensure information is easily accessible. Use clear section dividers and whitespace effectively.
- Incorporate subtle transitions and animations to enhance user experience, such as smooth scrolling and loading indicators.
